#  Smart EMI Planner

A modern, responsive web application to calculate EMI (Equated Monthly Installment) for loans instantly. Built with a clean UI and optimized user experience.

---

##  Live Demo

[Click Here to View Project](https://smart-emi-planner-ten.vercel.app/)

---

##  Features

* ✅ Calculate Monthly EMI instantly
* ✅ Displays Total Payment & Interest
* ✅ Smart suggestion for better planning
* ✅ Fully responsive (mobile + desktop)
* ✅ Clean and modern dark UI
* ✅ Fast and lightweight

---

##  Tech Stack

* React.js
* Tailwind CSS
* Vite

---

##  Deployment

This project is deployed on Vercel.
